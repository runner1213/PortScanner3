# Что это?
- Приложение на Java для удобного и быстрого сканирования портов на вашем сервере.

# Для чего это может понадобиться?
- Например, если вам нужно проверить доступность определённого сервиса на своём сервере.
- Убедиться, что нужные порты открыты для Minecraft, веб-сервера или других приложений.
- Найти закрытые или случайно открытые порты на своём сервере.

# Как использовать?
1. Запустите приложение через скомпилированный `.jar` в Releases. 

⚠ ВАЖНО! Убедитесь, что у вас установлена **JAVA 17 или выше**
2. В верхней части окна введите:
    - IP-адрес или доменное имя сервера в поле "IP / Домен".
    - Диапазон портов в формате `startPort-endPort`, например `1-65535`.
3. Нажмите кнопку **"Начать"**.
4. Введите количество потоков для сканирования в появившемся диалоговом окне *(чем больше потоков, тем быстрее сканирование, но выше нагрузка на сеть и компьютер)*.
5. Следите за результатами в консоли внизу окна:
    - Каждая строка показывает, открыт порт или закрыт.
    - В конце выводится **итоговый отчёт**: количество просканированных и открытых портов, список открытых портов.
6. Для повторного сканирования просто измените IP/диапазон портов и нажмите **"Начать"** снова.

# Особенности
- Поддержка многопоточного сканирования для ускорения проверки большого диапазона портов.
- Реальный вывод в консоль в режиме реального времени.
- Работает для TCP-портов, включая Minecraft, веб-серверы и другие сервисы.
